
# RGB

*Convert RGB colors to HSL.*

```JavaScript
import { fromRGB } from 'HSL'
```

<br>

## Parameter

*Supply the channels directly.*

```JavaScript
const hsl = fromRGB( R , G , B );
```

<br>

## Array

*Pass the channels as an array.*

```JavaScript
const hsl = fromRGB([ R , G , B ]);
```

<br>

## Alpha

*Optionally include an Alpha channel.*

```JavaScript
const hsla = fromRGB([ R , G , B , A ]);
```

```JavaScript
const hsla = fromRGB( R , G , B , A );
```
